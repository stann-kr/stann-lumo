# 기능 요구사항

---

## 공개 포트폴리오 페이지

| 경로       | 기능                                           |
| ---------- | ---------------------------------------------- |
| `/`        | 홈 — 네비게이션 카드 그리드, 타이핑 애니메이션 |
| `/about`   | About — 아티스트 정보, 동적 섹션(단락/철학)    |
| `/music`   | Music — 트랙 목록, 플랫폼 링크                 |
| `/events`  | Events — 공연 일정(Upcoming/Past), 이벤트 정보 |
| `/contact` | Contact — 연락처 정보, 방명록 폼               |
| `/link`    | Link — 외부 플랫폼 링크                        |

### 공통 기능

- 터미널 스타일 레이아웃 (사이드바 + 메인 콘텐츠)
- EN/KO 다국어 지원
- 동적 테마 색상 (CSS 변수 기반)
- 커서 글로우 효과
- 모바일 반응형

## 어드민 CMS

| 경로             | 기능                                          |
| ---------------- | --------------------------------------------- |
| `/admin`         | 로그인 페이지                                 |
| `/admin/home`    | 대시보드                                      |
| `/admin/about`   | 아티스트 정보 편집, 동적 섹션 CRUD (추가/삭제/타이틀·타입 변경/순서) |
| `/admin/music`   | 트랙 CRUD                                     |
| `/admin/events`  | 공연 일정 CRUD, RA API 연동, 이벤트 정보 편집 |
| `/admin/contact` | 연락처 정보 편집                              |
| `/admin/link`    | 링크 플랫폼 CRUD                              |
| `/admin/theme`   | 테마 색상 편집                                |

### 어드민 공통 기능

- EN/KO 탭 전환 편집
- 저장 알림 (3초 표시)
- 삭제 확인 모달
- 항목 순서 변경
- PAGE SETTINGS (각 페이지 타이틀/서브타이틀/섹션 타이틀 어드민 편집)
